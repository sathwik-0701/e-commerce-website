Project Overview

This is a full-stack E-commerce website built with a React + Vite frontend and a Node.js + Express backend. The application uses MongoDB to store user and product data, and integrates PayPal for secure payments.
Users can register, log in, log out, browse products, and make purchases.

Frontend (Client) :

The frontend is built with React JS and Vite for fast development.

How to Run (Client)

cd client
npm install
npm run dev

Backend (Server) : 

The backend is developed using Node.js, Express, and MongoDB.

How to Run (Server)

cd server
npm install
npm start
npm run dev
